      This educational project was implemented by me as part of my graduation from Tel-Ran School.
I used IntelliJ IDEA, Visual Studio Code and PostgreSQL to create it. Keeping in mind that my specialty is Java Backend Developer, the main part of this project is Backend. The meaning or idea of the project is to create a Web Application in which the user can register, add a post, edit a post, delete a post, write comments on posts, delete comments, edit user data.
      The backend is written in Java using the Spring Boot Framework and the MVC architectural pattern. Html code was written using Visual Studio Code, after which I used this code in Thymeleaf templates to display Frontend. The database was created in PostgreSQL and then connected to the project via: resources->application.properties with the appropriate settings. I used Liquibase to create tables in the database. And the interaction between entities and the database is carried out using JPA and the Hibernate framework. The project also contains an Open API which is implemented using the OpenAPI Generator and documented with the Swagger API Documentation. Testing done using H2 virtual database.

In conclusion, the programs and tools used in the project are as follows:

  1. IDE IntelliJ IDEA
  2. Visual Studio Code
  3. PostgreSQL
  4. Java
  5. Spring Boot Framework
  6. MVC architectural pattern
  7. HTML and CSS
  8. Thymeleaf
  9. PostgreSQL
  10. Liquibase
  11. JPA, Hibernate
  12. SQL
  13. Open API, Swagger
  14. H2 Data Base.
